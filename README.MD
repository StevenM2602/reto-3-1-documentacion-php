# Reto 3.1 – Documentación de proyecto PHP con PHPDocumentor

##  Descripción
Este proyecto corresponde al **Reto 3.1** de la asignatura **(Desarrollo Web en Entorno Servidor)**.  
El objetivo es documentar una pequeña aplicación desarrollada en PHP siguiendo el estándar PHPDoc y generar automáticamente una documentación en formato HTML utilizando **PHPDocumentor**.

La aplicación implementa un pequeño sistema de gestión de usuarios.



##  Estructura del proyecto

proyecto_documentacion/
├── index.php
├── Usuario.php
├── funciones.php
├── docs/
│ └── index.html
└── README.md
##  Tecnologías utilizadas

- PHP 8.x
- PHPDocumentor
- HTML5


##  Documentación

La documentación del proyecto ha sido generada automáticamente usando **PHPDocumentor**, a partir de los comentarios en formato DocBlock incluidos en el código fuente.

Para visualizarla:
1. Accede a la carpeta `docs/`
2. Abre el archivo `index.html` en tu navegador


##  Ejecución del proyecto

1. Abre el archivo `index.php` en un servidor local (XAMPP, WAMP, etc.)
2. Se mostrarán los datos de los usuarios por pantalla



##  Objetivo del reto

- Aplicar el estándar de documentación PHPDoc
- Generar documentación HTML automática
- Aprender a usar PHPDocumentor
- Documentar un proyecto PHP correctamente
